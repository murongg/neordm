NeoRDM 是一个现代化 Redis 桌面客户端，核心体验围绕 AI 辅助工作流设计。  
它帮助你更快理解 Redis 数据、分析键结构，并把自然语言需求转成带安全确认的 Redis 操作。

## AI 亮点

- AI 助手可感知当前连接、数据库、选中键和值上下文
- 支持自然语言生成 Redis 命令建议
- 支持解释键结构、TTL 与数据组织方式
- 对写入类或危险命令提供执行前确认
- 支持 OpenAI 兼容 API

## 核心功能

- 多连接 Redis 管理
- 支持树形键浏览、搜索、分组与数据库切换
- 支持 `string`、`hash`、`list`、`set`、`zset`、`stream`、`json` 的查看与编辑
- 内置 Redis CLI，支持历史记录和危险命令确认
- 支持主题、语言、隐私和工作区偏好设置
